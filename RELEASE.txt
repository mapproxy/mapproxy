Making a new MapProxy release
=============================

Preparation
-----------

Update version number in setup.py and commit and push.
Create tag with same version and push the tag.


Build and upload
----------------

The release on PyPI will be created automatically as will the docker images.


Announce
--------


- Add a new blog article to mapproxy.org with the most important changes
- MapProxy mailing list (copy from previous release mail as a template)
- Twitter (with link to blog)
